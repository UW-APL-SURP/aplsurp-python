APL SURP Python Course#

An introduction to Python in Oceanography for participants in the APL SURP (Summer Undergraduate Research) Program at the University of Washington Applied Physics Laboratory. This course will provide a foundation to the Python programming language and “ecosystem” specifically, and programming and open-source software more generally.

Goals#

  1. Introduce Python and its use in Oceanography through hands-on notebooks, to serve as a foundation for using Python in SURP research projects.

  2. Provide a ramp for future, self-guided learning to both strengthen core topics that were covered and explore new topics.

Course descriptions#

The structured part of the course consists of three 90-minute weekly lessons led by APL instructors (Ethan Campbell and Emilio Mayorga). The lessons will be in the form of interactive computational notebooks (“Jupyter” notebooks) running in Google Colab for convenience.

After the third week, we will hold weekly “office hours” for anyone to drop in and ask anything about Python or related topics.

This site also provides other materials for continued and deeper learning; see the Other resources section on the left. The content of this site, including the Jupyter notebooks used in the lessons, is open-source and available on GitHub at UW-APL-SURP/aplsurp-python.

Python is an open-source programming language. With open-source software, we are able to build on the work of others, contribute back to it, and build something new on top in collaboration with others.
